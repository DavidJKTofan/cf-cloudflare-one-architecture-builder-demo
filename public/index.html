The public Internet (HTTPS) also remains a viable connection option to the nearest Cloudflare data center.

Additional Notes:

  • Post-Quantum Cryptography (PQC) is supported.
  • Identity-based policies provide user-level control over access and security rules.
  • Mutual TLS (mTLS) and Service Tokens are typically used for service-to-service authentication and securing API endpoints.
  • IPSec/GRE Tunnels and CNI require Magic WAN or Magic Transit.
  • Public Peering or Private Network Interconnect (PNI) follows Cloudflare's open peering policy.
